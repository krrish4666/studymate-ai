Key Conventions
- Async everything: async def endpoints, AsyncSession, await db.execute()
- Services layer: Business logic in app/services/, route handlers in app/api/v1/
- Auth: JWT via Authorization: Bearer <token> header, injected via Depends(get_current_user)
- API prefix: All routes under /api/v1/
- API keys: User's Gemini key resolved from api_keys table (encrypted), falls back to empty string
- Testing: pytest with mocked AsyncSession + TestClient with dependency overrides
