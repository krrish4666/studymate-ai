About StudyMate AI HUB

Transforming how students learn with AI-powered tools

Our Mission

StudyMate AI HUB is an academic productivity web application that transforms raw study materials — including textbooks, PDFs, lecture slides, notes, and images — into structured, interactive learning formats. We leverage Google Gemini AI to generate notes, flashcards, mind maps, MCQ quizzes, and revision cheat sheets, making studying more efficient and effective.

Technology

Built with a Python/FastAPI backend and a vanilla HTML/CSS/JS frontend. Uses Google Gemini AI for content generation, PostgreSQL for data storage, and ReportLab for PDF generation. The application supports multiple file formats including PDF, DOCX, PPTX, TXT, and images.

Key Capabilities

  • AI-powered notes conversion with real-time streaming
  • Interactive flashcard generation and review
  • Configurable MCQ quizzes with instant scoring
  • Interactive mind map visualization
  • Concise revision cheat sheets
  • PDF export for all content types
  • Secure authentication with JWT and Google OAuth
  • Personalized theme system with 6 presets
